# My Coding Sandbox

Welcome to my collection of small, experimental, and hobby projects! This repository serves as a home for the tools and games I build in my free time to explore low-level programming, terminal graphics, and game logic.

## Projects Overview

|Project|Language|Platform|Status|
|---|---|---|---|
|**Minesweeper Clone**|C++|Windows|Completed|
|**Terminal Image Printer**|C|Linux|Active/Expanding|
|**Top-Down Shooter**|C|Linux|In Progress|
|**Music in Terminal**|Bash|Linux|Active/Expanding|
|**Toy CPU Emulator - Assembly**|C / Python|Active/Expanding|
## Detailed Project Breakdown

### 1.  Minesweeper Clone

A faithful terminal-based recreation of the classic Minesweeper game.

- **Language:** C++
- **Platform:** Windows (uses `conio.h` and Windows-specific terminal handling)
- **Current State:**  Plug and play!

### 2. Terminal Image Utility

A utility program that processes images and renders them directly into your Linux terminal using ANSI escape codes.

- **Language:** C
- **Platform:** Linux
- **Current State:** Basic image rendering, can handle multiple pictures in a directory

### 3.  Top-Down Terminal Shooter

An action game built entirely within the terminal buffer. Focuses on manual memory management and efficient terminal updates.

- **Language:** C
- **Platform:** Linux
- **Status:** **Under Construction**

### 4. Music in Temrinal

A utility to listen to music from youtube in the terminal based on fzf, mpv and yt-dlp. This is a simple program that I use instead of listening music in browser to save battery life!

- **Language** Bash
- **Platform** Linux
- **Status** Usable but needs set up the first time
 
### 5. Toy CPU and a Toy Assembler for the CPU

This is a simple project that that simulates a simple CPU with 256 bytes of memory and an Accumulator. I also Have a toy Assembler written in Python to play around with it.
In addition there is a utility program written in Python you can use to manually edit the memory of the Program (in Hexadecimals)

- **Language** C / Python
- **Platform** Linux
- **Status** Usable
